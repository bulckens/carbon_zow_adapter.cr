# carbon_zow_adapter.cr

A Zow Mailer API adapter for Carbon, Lucky's mail handler.

## Installation

Add the dependency to your `shard.yml`:

```yaml
dependencies:
  carbon_zow_adapter:
    github: bulckens/carbon_zow_adapter
```

Run `shards install`
